# 茶语

## Build Setup

```bash
# clone the project
git clone git@github.com:jieye12/tea.git

# enter the project directory
cd tea

# install dependency
npm install/pnpm install

# develop
npm run dev/pnpm dev
```

This will automatically open http://localhost:9528
